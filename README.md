# Sistema ERP Multi-Tenant

Sistema ERP completo com suporte a multiplas lojas onde cada loja possui:
- Painel administrativo completo
- Site publico proprio para clientes
- Banco de dados isolado por loja_id

## Tecnologias

### Backend
- Node.js
- Express.js
- PostgreSQL
- JWT
- Bcrypt
- Multer

### Frontend
- React.js
- Tailwind CSS
- React Router DOM
- Axios
- Recharts
- jsPDF

## Como Instalar

### Backend
1. Entre na pasta backend
2. Execute npm install
3. Copie .env.example para .env
4. Configure suas credenciais no .env
5. Execute npm run dev

### Frontend
1. Entre na pasta frontend
2. Execute npm install
3. Execute npm start
